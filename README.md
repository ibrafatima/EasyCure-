Doctor Appointment Management System A web-based application built using HTML, CSS, JavaScript, PHP, and SQL to simplify and manage healthcare appointments for both patients and doctors.

Features:

**Patient Appointment Booking Patients can register, browse available doctors, and book appointments based on availability and specialization.

**Doctor Portal Doctors can manage their schedules, view upcoming appointments, and update their availability through a dedicated interface.

**Admin Dashboard Administrators have full control over system users, appointment records, doctor management, and overall system maintenance.

**AI Chatbot Integration An intelligent AI-powered chatbot is integrated to analyze user-inputted symptoms and suggest possible diseases, enhancing preliminary diagnosis and guiding users to the right specialists.
